Detailed Description

存储SOAP中XML DOM的元素名以及数据的链表实现。

Detailed description starts here. 因为是顺序链表结构，插入性能上性能为O（N），但考虑数据量较小以及形成一 易懂的格式，仍然采用链表结构。

soapList所有节点指针，必须是calloc/malloc的。因为在soapList_isEmpty函数 函数中，会对所有节点进行free。
